## 一、市场定位输出模板

### 市场阶段表述

```text
当前市场属于{阶段}，上证/创业板/科创50呈现{趋势特征}。
位置上，指数距离{关键支撑/压力}较近，量能{放大/萎缩/持平}。
定性：{反弹/右侧/左侧/震荡}，操作上{谨慎参与/积极参与/小仓试探/观望}。
```

### 定性框架

| 定性类型 | 含义 | 操作指导 |
|---|---|---|
| 反弹 | 下跌途中的技术性修复 | 谨慎参与，注意风险 |
| 右侧 | 趋势确认后的机会 | 可以积极参与 |
| 左侧 | 提前布局，买在最低点前 | 风险较高，小仓试探 |
| 震荡 | 方向不明，上下空间均有限 | 控制仓位，高抛低吸 |

## 二、信号判断参考

### 见底信号检查清单

| 信号 | 检查点 | 参考阈值 |
|---|---|---|
| 补跌完成 | 前期强势板块是否补跌 | 核心高位板块风险释放 |
| 量能萎缩 | 成交额是否明显下降 | 低于近期均量约50%可视为地量参考 |
| 技术支撑 | 指数是否触及关键支撑不破 | MA60/MA120/前低支撑 |
| 情绪冰点 | 涨停少、跌停多、恐慌扩散 | 涨停低位、跌停显著增加 |
| 资金回流 | 北向/主力资金是否转向 | 连续净流入优于单日流入 |

### 信号数量与仓位参考

```text
满足0-1项：管住手，等待信号出现
满足2-3项：可以小仓试探，注意风险
满足4项以上：信号较充分，可以提高参与度
```

## 五、操作策略模板

### 三段式输出

```text
仓位建议：{0-2成/1-3成/3-5成/5-7成/7成以上}，理由是{市场状态+信号数量+外部风险}。
参与节奏：{低吸/观望/右侧确认/逢分歧布局/不追高}。
风险纪律：若{触发条件}，则{减仓/停止开仓/等待确认}。
```

### 仓位区间参考

| 市场阶段 | 仓位参考 | 节奏 |
|---|---:|---|
| 下跌中 | 0-2成 | 防守、等待 |
| 见底中 | 1-3成 | 小仓试探 |
| 震荡期 | 3-5成 | 高抛低吸 |
| 反弹中 | 5-7成 | 控制节奏，重视分歧 |
| 主升中 | 7成以上 | 顺势持有，动态止盈 |

## 六、事件时间线模板

> 边界：只写近期重大国内外事件日历，不写盘面复盘、北向资金、板块涨跌、指数涨跌。

### 已发生事件

```text
{日期}｜{标签}｜{事件名称}：{结果/结论}，对A股{风险偏好/流动性/主线板块}的影响为{影响}。
```

### 今日/未来事件

```text
{日期}｜{标签}｜{事件名称}：关注{核心变量}，若{情景A}则利好/利空{方向A}，若{情景B}则影响{方向B}。
```

### 事件示例

```text
6/18｜谨慎｜美联储主席/新主席首次公开讲话：关注降息路径和通胀表态，若偏鹰则压制成长估值，若偏鸽则利好科技成长。
6/20｜谨慎｜美国CPI公布：关注核心CPI环比，超预期会推升美元和美债利率，压制A股风险偏好。
6/24｜催化｜AI/半导体重大发布会：关注新产品、订单和资本开支指引，利好算力硬件、PCB、半导体设备链。
```

### 标签参考

| 标签 | 使用场景 |
|---|---|
| 催化 | 产业、政策、公司事件可能带来机会 |
| 利好 | 对市场或板块风险偏好有正面影响 |
| 偏多 | 影响偏正面但仍需验证 |
| 谨慎 | 存在风险、不确定性或潜在冲击 |

## 八、风险预警模板

| 风险类型 | 观察指标 | 应对 |
|---|---|---|
| 外部风险 | 美股大跌、VIX上升、汇率贬值 | 降仓或延后开仓 |
| 流动性风险 | 缩量、资金流出 | 谨慎参与 |
| 情绪风险 | 连板断裂、跌停增加 | 回避高位股 |
| 主线风险 | 核心板块补跌 | 等分歧结束 |
| 事件风险 | 数据/会议未落地 | 等结果再行动 |
