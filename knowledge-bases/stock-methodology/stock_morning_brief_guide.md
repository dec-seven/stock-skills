# 早报分析方法论指南

> **版本**: v1.6 | **最后更新**: 2026-06-18
> **定位**: A股早报生成的核心分析框架。主文件只保留规则、流程和决策标准；模板、样例表述、展开格式见 `stock_morning_brief_templates.md`。
> **原则**: 脚本管 FORMAT，LLM 管 INSIGHT；分析要可执行、可追踪、可迭代。

---

## 0. 核心原则

1. **先判断风险，再寻找机会**：早报优先识别隔夜风险、今日事件和仓位边界。
2. **先市场，后板块，再个股**：不在市场环境不支持时强行推荐方向。
3. **结论必须可执行**：每个判断都要落到仓位、节奏、方向或风险纪律。
4. **数据与观点分离**：数据来自脚本和行情源，观点来自方法论推理。
5. **择优迭代**：新样例更实用则吸收，旧规则不适用则降级或废弃。

## 2. 早报分析优先级

每日早报按以下顺序思考，避免泛泛而谈：

1. **隔夜外部风险**：美股、VIX、汇率、大宗、海外突发事件。
2. **昨日A股状态**：指数、成交量、涨跌家数、主线强度、资金流向。
3. **今日关键事件**：数据、会议、政策、产业催化、公司事件。
4. **市场阶段判断**：反弹、右侧、左侧、震荡、下跌。
5. **板块持续性**：主线是否延续，是否出现跷跷板或风格切换。
6. **操作策略**：仓位、节奏、低吸/观望/右侧确认。
7. **个股机会**：只在方向和节奏支持时给出。

## 3. 市场定位方法论

### 3.1 判断维度

| 维度 | 观察对象 | 结论落点 |
|---|---|---|
| 趋势 | 上证、创业板、科创50、均线系统 | 趋势向上/震荡/向下 |
| 位置 | 关键支撑压力、历史分位、前高前低 | 高位/中位/低位，风险收益比 |
| 阶段 | 反弹、修复、分化、加速、补跌 | 今日定性 |
| 节奏 | 连续阳阴线、量价、分时强弱 | 追随/等待/低吸 |

### 3.2 阶段定性

| 定性 | 含义 | 操作 |
|---|---|---|
| 反弹 | 下跌后的技术修复，趋势未确认 | 谨慎参与，不追高 |
| 右侧 | 趋势和资金确认 | 可提高参与度 |
| 左侧 | 提前试探，确定性不足 | 小仓试错 |
| 震荡 | 方向不明，板块轮动快 | 控仓，高抛低吸 |
| 下跌 | 风险释放未结束 | 防守，等待信号 |

### 3.3 关键点位带量突破确认

关键点位（整数关口、前高、平台压力位）附近的震荡，只有带量突破才视为有效确认。

```text
关键点位震荡 → 观察成交量是否配合 → 放量突破 = 方向确认 / 缩量突破 = 假突破风险
```

| 突破特征 | 含义 | 操作倾向 |
|---|---|---|
| 尾盘带量突破 | 资金主动进攻，确认度高 | 可提高关注度，等次日回踩确认 |
| 缩量突破 | 跟风不足，易回落 | 不追高，等放量或回踩支撑 |
| 突破后次日高开有量 | 资金承接 | 可顺势参与主线核心 |
| 突破后次日高开无量 | 获利盘兑现 | 先降预期，防冲高回落 |

输出要求：早报记者遇到指数或板块接近关键点位时，必须写清“突破是否带量”这一验证条件。

## 4. 信号判断方法论

### 4.1 核心信号

| 信号 | 判断重点 | 用途 |
|---|---|---|
| 补跌完成 | 前期强势方向是否补跌 | 判断风险释放 |
| 量能变化 | 放量突破、缩量调整、地量 | 判断承接和活跃度 |
| 技术支撑 | MA60/MA120/前低/平台位 | 判断防守边界 |
| 情绪温度 | 涨跌家数、涨停跌停、连板高度 | 判断赚钱效应 |
| 资金回流 | 北向、主力、成交额 | 判断参与意愿 |
| 主线持续 | 龙头强度、板块扩散、分歧修复 | 判断方向可靠性 |

### 4.2 信号数量与仓位

| 满足信号 | 市场含义 | 仓位倾向 |
|---|---|---|
| 0-1项 | 信号不足 | 0-2成，防守观望 |
| 2-3项 | 有试探价值 | 1-3成，小仓试错 |
| 4项以上 | 信号较充分 | 3-6成，结构性参与 |

### 4.3 开盘二次验证法

盘前判断只给方向预案，开盘后必须用竞价和10点量能做二次确认，避免外盘利好或昨日上涨后直接追高。

```text
隔夜外盘/宏观利好 → A股竞价强弱确认 → 10点成交量验证 → 决定追随、低吸或降仓
```

| 验证点 | 看什么 | 策略含义 |
|---|---|---|
| 集合竞价 | 指数高开幅度、主线龙头溢价、涨停/跌停家数 | 高开不等于强，弱竞价说明资金不认可 |
| 9:30-10:00 | 成交额放大、主线承接、冲高回落比例 | 有量可延续，无量易回落 |
| 10点节点 | 板块是否继续放量、核心股是否开板/跳水 | 判断主力锁仓还是出货 |
| 分歧承接 | 回落时是否缩量、低位是否补涨 | 判断结构性行情能否扩散 |

输出要求：早报中遇到“外盘反弹 + A股昨日已涨”的组合，必须写明“竞价 + 10点量能”确认条件。

## 5. 情绪温度分类（P0）

| 情绪状态 | 市场特征 | 仓位建议 | `SENTIMENT_CLASS` |
|---|---|---:|---|
| 高涨 | 普涨、放量、主线强、涨停扩散 | 5-7成 | `sentiment-hot` |
| 温和 | 结构性机会，赚钱效应一般 | 3-5成 | `sentiment-warm` |
| 寒冷 | 跌多涨少，资金谨慎，主线弱化 | 1-3成 | `sentiment-cold` |
| 极寒 | 恐慌杀跌，跌停增多，流动性差 | 0-2成 | `sentiment-frozen` |

情绪判断必须服务于仓位建议和心态管理，不能只做形容词描述。

## 6. 事件时间线方法论（P0）

### 6.1 定义边界

**重磅事件时间线 = 近期重大国内外事件日历**，不是盘面复盘、资金流向、板块涨跌或个股表现。

必须覆盖未来1-2周和刚发生的关键事件，核心是回答：哪些已发生/将发生的大事会影响A股风险偏好、流动性、主线板块和仓位节奏。

### 6.2 展示规则

1. **按时间排序**：已发生 → 今日 → 未来。
2. **最多10条**：优先保留未来1-2周确定日期的重大事件，已发生事件只保留仍有交易影响的结果。
3. **已发生事件必须给结论**：一句话说明结果、落地情况或市场反馈。
4. **未发生事件必须给关注点**：说明看什么变量、可能影响什么方向。
5. **标签必须明确**：催化、利好、偏多、谨慎等。
6. **禁止错放**：不得把昨日盘面、涨跌家数、北向流入、板块领涨、指数涨跌写进重磅时间线；这些内容属于昨日回顾/市场温度计/板块方向。

### 6.3 事件优先级

| 优先级 | 事件类型 | 示例 | 说明 |
|---|---|---|---|
| P0 | 全球央行/利率事件 | 美联储议息、主席/新主席首次公开讲话、点阵图、降息路径表态 | 直接影响全球流动性和风险偏好 |
| P0 | 核心宏观数据 | 美国CPI/PPI/非农/初请、PMI、国内LPR/社融/PMI/CPI/PPI | 影响利率预期、汇率、顺周期和成长估值 |
| P0 | 国内重大政策/会议 | 重要经济会议、金融论坛、监管政策、资本市场制度安排 | 影响A股方向、风险偏好和政策主题 |
| P1 | 全球/国内大型发布会 | 科技巨头发布会、AI/半导体/新能源重大发布、产业大会 | 影响主题催化和产业链映射 |
| P1 | 产业级催化 | 价格机制、订单、产能、行业会议、重要产品发布时间 | 影响主线板块持续性 |
| P2 | 龙头公司事件 | 业绩、订单、发布会、产能变动 | 仅保留能外溢到产业链的龙头事件 |
| 禁用 | 盘面/资金复盘 | 北向流入、板块涨跌、指数涨跌、成交额变化 | 不属于重磅时间线，转入对应模块 |

### 6.4 输出要求

- 已发生：`日期 + 标签 + 事件名称 + 结果/结论 + 对A股影响`
- 今日/未来：`日期 + 标签 + 事件名称 + 关注变量 + 潜在影响方向`
- 事件名称必须具体，不写“关注消息面”“海外扰动”等泛词。
- 若没有可靠日程，不编造；输出“待补齐重大财经日历”，并提示必须 WebSearch 验证。

## 7. 逻辑挖掘方法论

### 7.1 逻辑分类优先级

| 逻辑类型 | 持续性 | 判断标准 | 参与倾向 |
|---|---|---|---|
| 涨价逻辑 | 高 | 价格上涨可传导到业绩 | 优先 |
| 替代逻辑 | 高 | 国产替代、供应链重构、订单验证 | 优先 |
| 业绩线 | 高 | 业绩预增、订单增长、指引上修 | 优先 |
| 事件驱动 | 中 | 政策、会议、IPO、产业事件 | 看持续性 |
| 红利板块 | 中 | 高股息、低波动、防守属性 | 震荡防守 |
| 情绪修复 | 低 | 超跌反弹、短期修复 | 谨慎 |
| 题材炒作 | 极低 | 只有故事缺少数据 | 回避 |

### 7.2 板块方向输出规则

每个方向必须包含：

1. 方向名称
2. 核心催化
3. 持续性判断
4. 参与方式
5. 风险点

模板见 `stock_morning_brief_templates.md`。

### 7.3 强主线持仓行为识别

强主线不能只看概念强弱，要观察调整阶段的资金行为。

```text
强逻辑板块 → 调整不放大量杀跌 → 核心股高位锁仓/回踩加仓 → 主线仍有效
```

| 行为 | 含义 | 操作倾向 |
|---|---|---|
| 高点不卖 | 主力锁仓，趋势未结束 | 不因短线震荡轻易判死 |
| 跌了加仓 | 分歧中有资金承接 | 等回踩支撑低吸，不追冲高 |
| 调整量能健康 | 抛压未集中释放 | 维持主线观察 |
| 10点放量跳水 | 主力可能出货 | 降低板块优先级，回避高位股 |

适用方向：AI基建、光通信、半导体设备、存储扩产等中期逻辑明确但短线涨幅较大的方向。

### 7.4 板块反弹天数与节奏判断

连续反弹天数不是唯一标准，但可作为判断“启动/延续/分化/高抛”的辅助信号。

| 反弹天数 | 典型含义 | 操作倾向 |
|---|---|---|
| 1-2天 | 启动或超跌修复 | 观察次日资金是否持续 |
| 3天 | 趋势延续验证 | 可持有，但不宜再追高位 |
| 5天以上 | 分化风险加大 | 关注核心股，去弱留强 |
| 连续反弹后放量滞涨 | 短期顶部信号 | 减仓或高抛 |

判断时必须结合：主线逻辑强度、量能是否递减、核心股是否扩散。

### 7.5 强主线底部洗盘识别

强主线在主升浪前或中期，常出现底部缩量震荡，用于清洗浮筹。

```text
主线地位确认 + 底部缩量震荡 + 不跌破关键支撑 + 催化未兑现 = 洗盘概率高
```

| 行为 | 洗盘特征 | 出货特征 |
|---|---|---|
| 量能 | 缩量或温和震荡 | 放量杀跌 |
| 核心股 | 高位锁仓或回踩不破均线 | 高位破位、跳水 |
| 消息面 | 催化未落地或模糊 | 利好兑现、缺乏新催化 |
| 板块扩散 | 低位分支有补涨 | 跟风股全面回落 |

输出要求：早报判断“洗盘”时，必须给出支撑位置、催化节点和破位止损条件。

## 8. 主题催化与产业链映射方法论

### 8.1 学习素材处理边界

用户提供的盘面、消息、行业材料需先判定用途：

| 素材类型 | 处理方式 |
|---|---|
| 方法论学习素材 | 只抽取分析方法、判断框架、产业链映射逻辑，不直接写入报告 |
| 当日报告输入 | WebSearch验证后再进入事件、板块、策略和选股 |
| 选股素材 | 仅涉及具体股票筛选、评分、避坑的内容进入 `stock_selection_guide.md` |

学习素材必须区分“方法”与“事实”：事实数据未经验证不得成为报告结论。

### 8.2 新概念权威背书识别

当传统行业被重新命名，并与当前最强主线绑定，同时获得权威媒体、政策口径、产业资本或交易产品背书时，应提高其主题扩散权重。

```text
旧行业/传统资产 → 新概念命名 → 权威背书 → 市场共识形成 → 产业链重估
```

判断重点：

1. 新概念是否绑定当前主线，而不是孤立造词。
2. 背书来源是否具备传播和定价影响力。
3. 是否能解释上游、设备、材料、能源、连接等环节的重估。
4. 是否有资金承接或板块扩散迹象。

### 8.3 强主线上游重估法

分析强主线时，不只看最显性的设备、应用和龙头，还要沿产业链向上游和基础设施反推，寻找尚未充分定价的低位补涨方向。

```text
终端需求增长 → 基建/设备扩张 → 材料/能源/连接需求上升 → 上游资源或基础设施重估
```

适用场景：AI算力、新能源、低空经济、机器人、商业航天等中长期主线。

输出要求：早报中应写清“终端需求如何传导到上游”，不能只写概念名。

### 8.4 排产验证景气法

产业链景气判断不能只看价格和新闻，也要看排产、订单和淡旺季特征。

```text
排产同比高增 + 环比改善 + 淡季不淡 + 下游订单能见度提升 = 景气改善前置信号
```

判断标准：

| 信号 | 含义 |
|---|---|
| 同比高增 | 行业需求较去年明显改善 |
| 环比改善 | 短期趋势仍在向上 |
| 淡季不淡 | 需求强于季节性规律 |
| 订单能见度延长 | 从短期交易逻辑升级为中期景气重估 |

该方法用于盘前板块判断和策略，不直接等同于个股买入信号。

### 8.5 海外资金工具映射

海外市场围绕中国核心资产设计杠杆ETF、单票ETF或其他交易工具时，说明相关资产可能具备全球资金定价属性。

```text
海外资金工具出现 → 资金审美变化 → 全球资产定价属性提升 → A股同产业链映射
```

分析时关注“资金工具背后的审美”，而不是简单把ETF申报等同于利好。需观察是否能映射到A股产业链、龙头估值和主题扩散。

### 8.6 AI主线“存、算、连”拆解

AI产业链分析应拆为三条核心链路，并继续向上游基础环节延伸：

| 环节 | 观察方向 | 二级映射 |
|---|---|---|
| 存 | 存储、HBM、DRAM、SSD、服务器存储 | 存储芯片、材料、封装测试 |
| 算 | GPU、ASIC、AI服务器、液冷、电力 | 电力设备、铜铝钼等材料、散热 |
| 连 | 光模块、CPO、交换机、连接器、光芯片 | 光通信、PCB、高速连接、设备材料 |

使用该框架时，早报应判断当前资金炒作的是“存、算、连”哪一环，以及是否出现向上游或低位分支扩散。

### 8.7 科技细分精选四要素

泛科技板块内部差异大，需用四要素筛选可中期跟踪的核心标的。

| 要素 | 含义 | 判断重点 |
|---|---|---|
| 机构抱团 | 大机构持续持仓或加仓 | 龙虎榜、机构持仓变化、北向资金 |
| 高成长 | 业绩增速或订单能见度高 | 季报营收利润增速、行业排产 |
| 减持完成 | 主要股东减持窗口结束 | 减持公告、解禁进度、筹码锁定 |
| 筹码干净 | 浮筹较少，抛压可控 | 换手率、股东户数、套牢盘位置 |

使用规则：四要素满足越多，越适合作为中期配置；仅满足1-2项则按短线催化处理。

## 9. 外部关联方法论

### 9.1 外部因素分类

| 类型 | 观察重点 | A股映射 |
|---|---|---|
| 美股走势 | 三大指数、纳指、VIX、科技股 | 风险偏好、科技映射 |
| 汇率与美元 | 美元指数、离岸人民币 | 外资、成长股估值 |
| 大宗商品 | 原油、黄金、铜等 | 通胀、资源、避险 |
| 海外公司事件 | 订单、合作、业绩、产能 | 产业链映射 |
| 宏观数据 | CPI、PMI、非农、议息 | 流动性预期 |
| 重大突发 | 地缘、制裁、金融风险 | 风险偏好冲击 |

### 9.2 分析路径

```text
事件/数据 → 影响风险偏好/流动性/产业链 → 对A股指数和板块的影响 → 今日应对策略
```

### 9.3 外围反弹映射校准

外围反弹对A股不是简单“利好=上涨”，必须叠加A股前一日状态和量能环境校准。

| 外围变量 | A股前一日状态 | 早盘判断 |
|---|---|---|
| 美股科技反弹、VIX回落 | A股科技昨日已大涨 | 偏利好但不追高，等竞价和10点量能 |
| 地缘缓和、油价回落 | 指数缩量反弹 | 风险偏好修复，但持续性取决于成交额 |
| 海外AI链上涨 | A股AI链调整缩量 | 可关注回流，不追高位冲板 |
| 海外利好叠加A股放量突破 | 主线资金扩散 | 可提高仓位，优先核心分支 |

输出要求：外部利好必须落到“风险偏好、科技映射、通胀/加息预期、A股是否已提前反应”四个判断点。

## 10. 操作策略方法论

### 10.1 三段式输出

1. **今日仓位**：给出建议区间，并说明相比昨日是否加减仓。
2. **参与节奏**：低吸、观望、右侧确认、逢分歧布局、不追高。
3. **风险纪律**：明确什么情况减仓、停止开仓或等待确认。

### 10.2 仓位参考

| 市场阶段 | 仓位参考 | 操作节奏 |
|---|---:|---|
| 下跌中 | 0-2成 | 防守等待 |
| 见底中 | 1-3成 | 小仓试探 |
| 震荡期 | 3-5成 | 高抛低吸 |
| 反弹中 | 5-7成 | 等分歧，控节奏 |
| 主升中 | 7成以上 | 顺势持有，动态止盈 |

### 10.3 涨后不追高规则

当A股昨日已明显上涨，次日即使外围偏利好，也不得直接给“高开高走”判断。

```text
昨日大涨 → 今日外部利好 → 不追开盘高点 → 等分歧/回踩/量能确认
```

| 场景 | 参与方式 |
|---|---|
| 高开有量，主线核心不跳水 | 持有或小幅跟随核心，不扩散追杂毛 |
| 高开无量，10点前冲高回落 | 减少开仓，等待午后承接 |
| 平开强承接，主线低位补涨 | 可小仓参与补涨或回踩支撑股 |
| 板块短线高位放量滞涨 | 先防出货，等回踩支撑再看 |

### 10.4 震荡市区间操作策略

震荡市的核心不是预测方向，而是控制仓位和节奏。

| 原则 | 具体动作 | 反面动作 |
|---|---|---|
| 不满仓 | 保留现金应对波动 | 一次性打满 |
| 涨减 | 反弹到压力区或放量滞涨时减仓 | 追涨加仓 |
| 跌吸 | 回踩关键支撑且缩量时分批低吸 | 恐慌杀跌 |
| 情绪隔离 | 不因单日涨跌改变计划 | 被市场情绪牵着走 |

输出要求：震荡市早报必须写明“仓位上限、高抛低吸区间、情绪管理提醒”三点。

## 11. 风险预警方法论

| 风险类型 | 观察指标 | 应对 |
|---|---|---|
| 外部风险 | 美股大跌、VIX上升、汇率贬值 | 降仓或延后开仓 |
| 流动性风险 | 缩量、资金流出 | 谨慎参与 |
| 情绪风险 | 连板断裂、跌停增加 | 回避高位股 |
| 主线风险 | 核心板块补跌 | 等分歧结束 |
| 事件风险 | 数据/会议未落地 | 等结果再行动 |

风险预警必须写清楚“触发条件”和“应对动作”。
