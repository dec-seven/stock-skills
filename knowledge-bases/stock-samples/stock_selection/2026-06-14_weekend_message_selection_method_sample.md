## 原始素材摘录

素材围绕三类行业线索：

1. 算力金属：AI算力基建对上游关键有色金属的重定价，并获得权威背书。
2. 锂矿：排产同比高增、环比改善、淡季不淡、储能订单能见度延长、价格预期上修。
3. 光通信：海外围绕中国核心资产的单票2倍杠杆ETF申报，反映AI全球竞赛中“存、算、连”环节出现更多中国公司身影。

## 已提炼进选股方法论

写入文件：`knowledge-bases/stock-methodology/stock_selection_guide.md`

提炼内容：
1. 产业链映射选股：终端需求 → 产业链增量环节 → 价格/订单/排产变化 → 业务纯正个股。
2. 排产/订单验证选股：排产同比高增、环比改善、淡季不淡、订单能见度延长时，优先筛选利润弹性最大、传导路径最直接的公司。
3. 全球资金定价选股：海外杠杆ETF/单票ETF等工具不直接构成买入理由，应判断公司是否具备全球竞争力、稀缺性、资金容量、核心产业位置和合适技术位置。
4. 学习素材股票不直接复用：素材股票/方向 → 提炼选股逻辑 → 候选池扩展 → 三层映射法评分 → 技术面评分 → 排序择优。

## 评分体系映射

| 评分项 | 从素材提炼的加分条件 |
|---|---|
| 业务纯正度 | 主营直接对应增量环节，而非弱概念关联 |
| 行业地位 | 资源储量、产能规模、技术壁垒或客户地位靠前 |
| 涨价受益度 | 产品价格、资源品价格或订单价格可传导至利润 |
| 业绩验证 | 排产、订单、价格、财报已有至少一项验证 |
| 催化剂临近 | 权威背书、产业数据、价格预期、订单落地在近期出现 |
| 特殊标签 | 新概念核心映射、全球资金定价、国产替代稀缺等 |

## 注意

本素材只用于沉淀选股规则。素材中涉及的行业方向、潜在个股或数据，不得直接进入早报推荐池或股票跟踪池，必须重新经过候选池扩展、三层映射法评分和技术面评分。
